隐私声明

  • 本地优先 BlackEagle 在用户浏览器内工作，不运营自有服务器，不采集、不出售个人数据，亦不注入广告。
  • 数据留在本机 网页内容解析、自动化操作及模型请求均在本地执行；配置、对话与偏好保存在浏览器本地存储，可随时清理。
  • 模型调用 只有在你主动配置自己的 LLM Endpoint / API Key 时，相关内容才会直接发送至你指定的第三方模型服务；我们不保存、不代理这些请求。
  • 下载与通知 由你触发的导出会写入本地文件；通知与提醒仅用于任务状态提示或定时提醒，不含广告推送。
  • 卸载与删除 在 chrome://extensions 中移除 BlackEagle 即可停用扩展；如需彻底清理，可在移除前在扩展详情页选择“清除数据”。

权限用途

  • <all_urls> + activeTab + tabs + scripting 读取并操作当前页面，这是实现“阅读页面并自动点击/填写”的核心能力，操作仅在你主动触发时执行。
  • storage 将模型 Endpoint、API Key（本地加密存储）与个人偏好保存在浏览器内，方便下次使用。
  • alarms + notifications 支持定时任务与提醒，例如页面处理完成或计划任务的提示。
  • downloads 在你选择导出摘要或文件时，将结果保存到本地电脑。
  • sidePanel 在浏览器侧边栏中呈现 UI，便于在任意页面开启 AI 助手。
