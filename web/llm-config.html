大模型配置类型说明

BlackEagle 支持多种类型的大模型能力，并会根据不同功能自动调用对应的模型类型。目前插件可识别并使用以下 4 类大模型：

对话模型（Chat / LLM）

用于日常对话、网页内容理解、文本生成与推理等核心交互能力。

文档识别模型（Document / File Understanding）

用于解析和理解网页中的长文本、结构化文档或用户上传的文件内容。

图片生成模型（Image Generation）

用于根据文本指令生成图片，支持创作、设计和内容增强等场景。

视觉识别模型（Vision / Multimodal）

用于理解网页中的图片、截图或视频画面，实现真正的多模态分析能力。

插件会根据功能场景自动匹配并调用对应类型的大模型。

如果你希望获得最完整、最强大的 AI 浏览器体验，建议将以上四类模型全部配置完成。

重要提示：BlackEagle 仅支持符合 OpenAI SDK 规范的模型接口。只有遵循该规范的大模型，插件才能正确识别、调度并稳定运行相关功能。
