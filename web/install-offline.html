Chrome / Edge 离线安装教程（ZIP 包）

无需商店，直接下载 BlackEagle.zip 解压并在浏览器加载已解压的扩展。

下载 BlackEagle.zip 返回下载区

Chrome 安装步骤（ZIP）

  1. 下载并解压 BlackEagle.zip 到任意本地文件夹。
  2. 在地址栏输入 chrome://extensions，开启「开发者模式」。
  3. 点击「加载已解压的扩展程序」，选择刚解压的文件夹。
  4. 加载成功后，固定 BlackEagle 图标到浏览器工具栏即可使用。

更新到新版本

下载新 ZIP，解压到新文件夹，在扩展页点击「移除」旧版本，再重新「加载已解压的扩展程序」。

Edge 安装步骤（ZIP）

  1. 下载并解压 BlackEagle.zip。
  2. 在地址栏输入 edge://extensions，开启「开发人员模式」。
  3. 点击「加载已解压的扩展」，选择解压后的文件夹。
  4. 安装完成后，可在工具栏固定 BlackEagle 以便快速使用。

常见问题

  • 提示无法加载？确认选择的是解压后的文件夹根目录。
  • 按钮灰色？确保已打开「开发者模式 / 开发人员模式」。
  • 升级后报版本冲突？先移除旧版本再重新加载新文件夹。

快速核对

ZIP 已解压到本地，并且选择的是解压后的文件夹。

浏览器扩展页已开启「开发者模式」。

安装后可在设置里填入你的 LLM Endpoint + API Key 完成配置。
