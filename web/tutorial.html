DeepSeek 基础配置教程

按照下面步骤，完成 API Key 申请与在 BlackEagle 的基础对话配置。

一、申请 DeepSeek API Key

  1. 访问 DeepSeek 官网（开发者门户），注册并登录账号。
  2. 打开「API Keys」页面，点击新建，生成一个 API Key。
  3. 复制并妥善保存该 Key，用于 BlackEagle 的模型配置。

常用 Endpoint 示例

例如：https://api.deepseek.com/v1（具体以官方文档为准）

二、在 BlackEagle 中进行配置

  1. 打开插件侧边栏，进入「设置 / 模型配置」页面。
  2. 填写 Endpoint、选择模型（如 deepseek-chat），粘贴 API Key。
  3. 保存后，在对话框输入「Hello」进行测试。

常见问题

无法连接？请检查 Endpoint 是否正确、Key 是否有效，以及是否有网络代理限制。

模型返回慢？尝试降低温度、选择更快的模型，或优化提示词长度。
