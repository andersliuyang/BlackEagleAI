下载与安装

BlackEagle 支持 Chrome 与 Edge。当前版本提供 GitHub Releases 下载。

下载 Chrome 版 下载 Edge 版
Chrome 安装：
  1. 打开 chrome://extensions 并开启「开发者模式」。
  2. 点击「加载已解压的扩展程序」，选择插件解压后的文件夹。
Edge 安装：
  1. 打开 edge://extensions 并开启「开发人员模式」。
  2. 点击「加载已解压的扩展」，选择插件解压后的文件夹。

提示：建议优先使用你自己的 LLM Endpoint + API Key，获得更快、更安全、更个性化的体验。
