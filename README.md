## 🔐 Privacy & Security

Our core design principles are **data sovereignty** and **local-first**.

- **No Backend Servers**: The desktop client and browser extension run locally and do not rely on any centralized servers.
- **We Don't Touch Your Data**: Your files, web content, and AI requests are sent directly by you to the model provider you configure. We do not proxy or log them.
- **You Are in Control**: You are free to choose any language model and have full control over your API costs.

## Large Model Configuration Types

For the most complete AI experience, we recommend configuring the following types of models. BlackEagle will automatically call the most appropriate model for the task scenario:

- **Chat / LLM**: For dialogue, reasoning, and text generation.
- **Document / File Understanding**: For parsing long texts or structured files.
- **Image Generation**: For generating images from text instructions.
- **Vision / Multimodal**: For understanding image or video content.

**Note**: BlackEagle only supports model interfaces that are compatible with the OpenAI SDK specification.
